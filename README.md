## Introduction
Pegasus is a fully functioning agentic harnessa and runtime for performing general purpose digital and physical tasks. It is similar to existing coding CLI agents but extends them by focusing on long-running tasks. Pegasus has all the features of a regular coding agent and while extending to other tasks such as browsing, video etc. Pegasus treats open source models as first class citizens and aims to be the best agentic harness for open source AI models. 

Set your open router api key
`export API_KEY=sk-xxx`

Run pegasus with the following command
`pip install -r requirements.txt && python main.py`
